# LeadIQ — Lead Conversion Predictor

ML-powered lead scoring system that predicts the probability of a lead converting to a customer.

## Live Demo
🔗 https://leadiq-m.streamlit.app/

## Tech Stack
- Python, scikit-learn, Streamlit
- Logistic Regression + Random Forest ensemble
- ROC AUC: 0.899 | Accuracy: 83%
